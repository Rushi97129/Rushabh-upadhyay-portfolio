# Rushabh-upadhyay-portfolio

Simple Bootstrap portfolio starter. Deploy via GitHub Pages.

## How to use

1. Replace placeholder images in `assets/images/`.
2. Update content in `index.html`.
3. Commit and push to GitHub.
4. Enable GitHub Pages (branch: `main` / folder: `/ (root)`).

## Tech
- Bootstrap 5 (CDN)
- Optional: Bootstrap Icons
